# myweb
自己创建的小型博客系统

### 使用的主要技术栈有：
> * nodejs
> * express
> * mongodb


### Build Setup
``` bash
# clone the repository to your local file
git clone git@github.com:smbera/myweb.git

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
